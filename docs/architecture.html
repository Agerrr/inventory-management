Key Patterns

Reactivity

Raw data stored in ref() variables. Derived/filtered data computed via computed(). Filter changes trigger watch() callbacks that re-fetch from API. Template automatically re-renders.

Filter System

4 global filters (Time Period, Warehouse, Category, Order Status) managed by singleton useFilters() composable. Passed as query params to backend. 'all' value means no filter applied.

Modal Pattern

Click table row to set selectedItem ref and open modal. Modals use Teleport to body with Transition animations. Props down (:item), events up (@close).
